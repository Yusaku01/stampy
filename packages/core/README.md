# @stampy/core

Framework-agnostic SVG stamp rendering, project JSON, Web Components, and press
animations for static sites and web apps.

## Install

After the first npm release:

```sh
pnpm add @stampy/core
```

Until then, use the repository workspace or the packed tarball generated by
`pnpm pack:core`.

## Web Component

```ts
import {
  installStampyAnimations,
  pressStamp,
  registerStampyElements,
} from "@stampy/core";
import "@stampy/core/styles.css";

registerStampyElements();
installStampyAnimations();

const stamp = document.querySelector("stampy-mark");
await pressStamp(stamp, { preset: "classic" });
```

```html
<stampy-mark
  main-text="Visited"
  sub-text="Local Guide"
  date-text="2026.06.27"
  shape="circle"
  icon="steam"
  ink="#b93632"
  paper="#fffaf4"
  distress="0.72"
  roughen="2.25"
  rotate="-6"
></stampy-mark>
```

Supported shapes are `circle`, `ticket`, and `rounded`. Supported icons are
`steam`, `star`, `spark`, and `none`.

## Render And Export

```ts
import {
  downloadStampPng,
  downloadStampSvg,
  renderStampSvg,
  stampPngBlob,
  stampSvgBlob,
} from "@stampy/core";

const design = {
  mainText: "Visited",
  subText: "Local Guide",
  dateText: "2026.06.27",
  ink: "#b93632",
};

const svg = renderStampSvg(design);
const svgBlob = stampSvgBlob(design);
const pngBlob = await stampPngBlob(design, 1024);

downloadStampSvg(design, "stamp.svg");
await downloadStampPng(design, "stamp.png", 1024);
```

`stampPngBlob` and `downloadStampPng` use browser image decoding and Canvas, so
call them in a browser-like runtime.

## Project JSON

Use Stampy Project JSON when a site or app needs to store editable stamp data
instead of a flattened image.

```ts
import {
  createStampyProject,
  parseStampyProject,
  serializeStampyProject,
} from "@stampy/core";

const project = createStampyProject({
  name: "Reward stamp",
  locale: "en",
  theme: "auto",
  stamp: {
    mainText: "Reward",
    subText: "Coffee Club",
  },
  animation: {
    preset: "ink",
    durationMs: 720,
  },
});

localStorage.setItem("stampy-project", serializeStampyProject(project));

const restored = parseStampyProject(
  localStorage.getItem("stampy-project") ?? "",
);
```

Locales are `en` and `ja`. Themes are `auto`, `light`, and `dark`.

## Press Animations

Built-in presets are `classic`, `bounce`, `ink`, and `none`.

```ts
await pressStamp(stamp, {
  preset: "bounce",
  durationMs: 820,
  intensity: 1,
});
```

Pass an `animator` when the host app wants to own the motion fully:

```ts
await pressStamp(stamp, {
  preset: "classic",
  durationMs: 900,
  animator: async (element, context) => {
    const animation = element.animate(
      [
        {
          opacity: 0,
          transform: "translateY(-48px) scale(1.55) rotate(-12deg)",
        },
        { opacity: 1, transform: "translateY(0) scale(1) rotate(-3deg)" },
      ],
      {
        duration: context.durationMs,
        easing: "cubic-bezier(.18,.86,.2,1)",
        fill: "both",
      },
    );

    await animation.finished;
  },
});
```
