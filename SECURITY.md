# Security Policy

Stampy is currently pre-1.0 and has no private disclosure process yet.

Please report suspected security issues by opening a GitHub security advisory
after the public repository is created. Until then, do not publish exploit
details in a public issue.

## Scope

Security-sensitive areas include:

- project JSON parsing,
- SVG generation and text escaping,
- file import/export in Studio,
- and any future hosted gallery or cloud sync features.

Stampy does not currently include accounts, server-side storage, or networked
sync.
